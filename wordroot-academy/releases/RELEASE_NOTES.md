# Release Archive — v1.0.0

This folder contains archived snapshots of each released version of index.html.

## How to use

To view any past version, open the corresponding HTML file directly in a browser.

To roll back to a previous version:
```bash
cp releases/v1.0.0.html index.html
git add index.html
git commit -m "revert: roll back to v1.0.0"
git push
```
